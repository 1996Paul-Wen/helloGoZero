# 1. 安装docker

# 2. 启动mysql容器
## 拉取镜像
```
docker pull mysql:latest
```

## 启动mysql
```bash
docker run -d \
  --name mysql8 \
  -p 3306:3306 \
  -e MYSQL_ROOT_PASSWORD=abcd \
  -v ~/mysql_data:/var/lib/mysql \
  mysql:latest
```

## 登陆mysql验证
`docker exec -it mysql8 mysql -uroot -p`

# 3. 启动es容器
## 拉取镜像
```bash
docker pull docker.elastic.co/elasticsearch/elasticsearch:7.17.29
```

## 启动es容器
```bash
docker run -d \
  --name elasticsearch \
  -p 9200:9200 -p 9300:9300 \
  -e "discovery.type=single-node" \
  -e "ES_JAVA_OPTS=-Xms512m -Xmx512m" \
  -v ~/es_data:/usr/share/elasticsearch/data \
  -v ~/es_logs:/usr/share/elasticsearch/logs \
  -v ~/es_plugins:/usr/share/elasticsearch/plugins \
  docker.elastic.co/elasticsearch/elasticsearch:7.17.29
```

## 验证
```bash
curl -X GET "http://localhost:9200"
```

# 启动服务
```bash
# 运行容器
docker run -d \
  -p 8888:8888 \
  -v /local/path/to/safebox-api.yaml:/root/etc/safebox-api.yaml \
  --name safebox \
  safebox-service
```

## 验证
```bash
curl -i http://localhost:8888/greet/from/you
```
